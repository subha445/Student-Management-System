# 🎓 Student Management System

A simple web-based Student Management System built using **HTML, CSS, and JavaScript**.  
This project allows users to add, view, and delete student details in a structured table format.

---

## 🚀 Live Demo
👉 https://subha445.github.io/Student-Management-System/

---

## ✨ Features

- ➕ Add student details (Name, Roll Number, Marks)
- 📋 Display data in a table format
- ❌ Delete student records
- ⚡ Instant updates using JavaScript
- 💾 Data persistence using LocalStorage
- 📱 Simple and responsive UI

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## 💡 How It Works

- User enters student details in input fields  
- Clicks "Add Student" button  
- Data is stored in array and saved using LocalStorage  
- Table updates automatically  
- Delete button removes specific student record
